# SAL-2628: Endpoint docker-compose and bootstrap

## Target paths
- deploy/endpoint/docker-compose.yml
- deploy/endpoint/.env.template
- deploy/endpoint/README.md
- deploy/endpoint/bootstrap.sh

## Acceptance criteria
```
clean Debian VM, paste registration_token, endpoint registers + heartbeats.
```

## Verification approach
- Execute `bootstrap.sh` on a fresh Debian VM.
- Fill in the `REGISTRATION_TOKEN`.
- Verify the service starts via `docker compose ps`.
- Use `mcctl endpoint list` to confirm registration and heartbeat reporting.

## Risks
- Operator may forget to set the registration token.
- Network connectivity to hub must be allowed outbound on port 443.
