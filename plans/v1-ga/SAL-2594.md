# SAL-2594: mc-init soulkey minting and allowlist registration

## Target paths
- deploy/appliance/mc-init/mint_soulkeys.sh
- deploy/appliance/mc-init/register_allowlist.sh
- deploy/appliance/mc-init/README.md

## Acceptance criteria
- 6 soulkeys generated idempotently; `_soulkeys` rowcount=6; `_soulkey_allowlist` for coo has ≥4 rows

## Verification approach
Run the container startup; verify that the six key files exist in `/state/secrets/`. Query the PostgreSQL tables to confirm row counts.

## Risks
- Idempotency must be preserved; scripts use existence checks and ON CONFLICT DO NOTHING.
- Ensure DB connection credentials are correct in the container environment.
