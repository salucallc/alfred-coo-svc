# SAL-2592: Split Docker Networks

## Target paths
- deploy/appliance/docker-compose.yml
- deploy/appliance/tiresias/network_split.md

## Acceptance criteria
- `docker exec alfred-coo curl --max-time 5 https://api.github.com` fails (no internet route).
- `docker exec mcp-github curl --max-time 5 https://api.github.com` succeeds (egress allowed).

## Verification approach
- Run the above curl commands inside the respective containers and check exit codes.

## Risks
- R1: If `internal:true` breaks DNS, fallback to iptables OUTPUT REJECT in init container (depends on TIR-09).
