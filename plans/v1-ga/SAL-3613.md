## Acceptance criteria
**Acceptance:**

1. Plugin installs via pip + passes SDK contract suite (W1-C).
2. Reference LangChain agent (a simple ReAct agent with one tool) registers successfully and dispatches a task end-to-end.
3. Audit events emit for: dispatch received, tool calls inside the LangChain run, dispatch complete.

## Verification approach
Run the full pytest suite (`pytest -q`) and perform a manual end‑to‑end smoke test registering a dummy LangChain agent and dispatching a task, confirming that the `tool_call` audit event appears in the plugin's audit log.

## Risks
- Dependency version mismatch with the pinned LangChain 0.1.x range.
- Absence of a real LangChain CLI in the test environment (simulated here).
